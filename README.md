# chat-ui-design
A responsive Chat Application UI built using HTML and CSS that simulates a modern messaging interface with user and bot messages.

A modern and responsive Chat Application UI built using HTML and CSS.

## 🚀 Features

* Clean modern interface
* User & Bot message styling
* Online status indicator
* Responsive layout

## 🛠️ Technologies Used

* HTML
* CSS

## ▶️ How to Run

Simply open `index.html` in your browser.
